## Pipeline Overview

This release contains the complete output of a 7-sprint synthetic maternity
research pipeline that develops and validates a qualitative research instrument
for exploring maternity care experiences.

### Sprint History

| Sprint | Module | Description |
|--------|--------|-------------|
| 1 | Data Ingestion | Synthea EHR parser, FinePersonas filter, SNOMED registry |
| 2 | Persona Construction | 150 EHR-grounded composite personas with enriched narratives |
| 3 | Questionnaire Generation | 5 versioned instruments with EHR adaptation |
| 4 | Synthetic Interviews | 300 BIBD sessions with multi-provider persona rotation |
| 5 | Evaluation | 6-component quality assessment (richness, coverage, inter-rater, SERVQUAL) |
| 6 | Refinement | Blind spot analysis, probe enrichment, robustness testing, saturation |
| 7 | Manuscript & Release | Supplementary materials, expert evaluation, release packaging |
